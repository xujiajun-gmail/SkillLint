# Example Skills Collections

This directory contains public agent skill samples collected for SkillLint testing and evaluation.

## Collections

### 1. General collection
- path: `examples/`
- index: `examples/index.json`
- size: 100 skills
- sources: OpenAI, Vercel, antfu, Apify, Callstack, Obsidian, and other public repositories

### 2. Chinese community collection
- path: `examples/zh-community/`
- index: `examples/zh-community/index.json`
- size: 100 skills
- sources: Chinese community repositories such as `tanweai/pua`, `xstongxue/best-skills`, `xu-xiang/everything-claude-code-zh`, `partme-ai/full-stack-skills`, `shareAI-lab/shareAI-skills`, and `KKKKhazix/khazix-skills`
