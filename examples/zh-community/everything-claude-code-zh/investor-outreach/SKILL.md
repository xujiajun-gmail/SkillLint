# 投资者外联（Investor Outreach）

编写简短、个性化且易于跟进的投资者沟通内容。

## 何时激活（When to Activate）

- 向投资者发送冷启动邮件（Cold Email）时
- 撰写请求引荐（Warm Intro）的简述时
- 在会议后或未收到回复时发送跟进（Follow-up）邮件时
- 在融资过程中编写投资者更新（Investor Updates）时
- 根据基金投资理念（Thesis）或合伙人匹配度定制外联内容时

## 核心规则（Core Rules）

1. **个性化**：每条发出的信息都必须是个性化的。
2. **低摩擦**：让读者的回应或行动尽可能简单直接。
3. **数据说话**：多用事实（Proof）证明，少用形容词堆砌。
4. **保持简洁**：言简意赅。
5. **拒绝模版**：严禁发送可以发给任何投资者的通用副本。

## 冷启动邮件结构（Cold Email Structure）

1. **主题行（Subject Line）**：短促且具体
2. **开场白（Opener）**：为什么要找这位特定的投资者
3. **推介（Pitch）**：公司是做什么的，为什么是现在，有哪些核心数据/证明点
4. **诉求（Ask）**：一个具体的下一步行动
5. **落款（Sign-off）**：姓名、职位，必要时加上一个信誉背书点

## 个性化来源（Personalization Sources）

参考以下一项或多项内容：
- 相关的投资组合公司（Portfolio Companies）
- 公开的投资理念、演讲、帖子或文章
- 共同的人脉关系
- 与投资者关注领域明确的市场或产品匹配度

如果缺失这些上下文，请向用户索取，或声明该草稿是等待个性化填充的模板。

## 跟进节奏（Follow-Up Cadence）

默认建议：
- 第 0 天：首次外联
- 第 4-5 天：简短跟进，提供一个新数据点
- 第 10-12 天：最后一次跟进，利落地结束

除非用户要求更长的序列，否则在此之后不要继续催促。

## 引荐请求（Warm Intro Requests）

为引荐人提供便利：
- 说明为什么这次引荐是匹配的
- 提供一段可直接转发的简介（Forwardable Blurb）
- 确保转发简介字数在 100 字以内

## 会后更新（Post-Meeting Updates）

包含以下内容：
- 讨论的具体事项
- 承诺的回答或更新
- 若有，提供一个新的数据证明点
- 下一步计划

## 质量把控（Quality Gate）

在交付前自检：
- 信息是否已个性化？
- 诉求（Ask）是否明确？
- 是否没有废话或带有乞求色彩的语言？
- 数据证明点是否具体？
- 字数是否精炼？
