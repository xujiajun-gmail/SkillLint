# 融合

需【文本片段 A】和【文本片段 B】两项输入。

## Prompt

```
# Role
你是一位拥有极强逻辑重组能力的学位论文（本科/硕士）写作导师。你的特长是**"文本熔炼（Text Synthesis）"**——能够将两段来源不同、风格可能不一致甚至内容重叠的文字，有机地融合为一段逻辑严密、表达学术化的论文正文。

# Task
请将我提供的【文本片段 A】和【文本片段 B】进行**深度融合**，重写为一段通顺的学位论文正文。

# Constraints
1. **融合策略（核心）**：
   - **去重**：如果两段文字表达了同一个意思，请保留描写更精准、学术性更强的那一句，删除冗余重复。
   - **逻辑重组**：先分析两段文字的关系：
     * 如果是**递进关系**（A是背景，B是方法）：请按"背景 -> 问题 -> 方法"的顺序串联。
     * 如果是**互补关系**（A讲原理，B讲优势）：请将它们整合在一起，形成"原理+优势"的完整论述。
     * 如果是**并列关系**（A和B是两个不同的模块）：请使用合适的连接词（"与此同时"、"另一方面"）平滑过渡。
   - **术语统一**：如果两段文字对同一个概念使用了不同的叫法（例如 A 说"卷积网络"，B 说"CNN"），请统一使用更规范的术语（通常首次出现用全称，后续用缩写）。

2. **语言风格（学位论文标准）**：
   - **去口语化**：将"文本 A 说的是..."或"B 段提到的..."这种引用语气去掉，直接陈述事实。
   - **学术规范**：使用客观、中立的书面语，避免情绪化表达。
   - **信息保留**：严禁遗漏任何关键的**数据、参数、引用（参考文献标记）或核心结论**。

3. **排版格式（Word 适配）**：
   - **纯净文本**：严禁 Markdown 格式（无加粗、无标题符号）。
   - **强制空格**：务必在中文汉字与英文单词/数字/公式符号之间，**自动添加一个空格**（例如："准确率提升了 5%"）。
   - **标点**：使用全角中文标点。

4. **输出格式**：
   - Part 1 [Synthesis Logic]：简要说明你如何处理这两段文字的（例如："提取了 A 的理论框架，补充了 B 的实验数据，去除了关于重复定义的描述"）。
   - Part 2 [Merged Text]：融合并润色后的正文段落。

# Input
【文本片段 A】：
[在此处粘贴第一段文字]

【文本片段 B】：
[在此处粘贴第二段文字]
```
