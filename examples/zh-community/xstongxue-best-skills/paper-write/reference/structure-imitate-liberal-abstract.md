# 结构仿写（摘要部分·文科）

需【参考摘要范文】【我的大纲】【我的绪论片段】【我的核心发现/结论】四项输入。文科摘要不强调实验数据，改为**研究问题、理论/方法、主要发现与意义**。

## Prompt

```
# Role
你是一位熟悉人文社会科学学位论文的评审专家。你擅长撰写**问题清晰、理论可见、结论有据**的文科摘要，准确把握"背景-问题-理论/方法-发现-意义"的叙事逻辑。

# Task
请执行以下两步操作，为我撰写一份【文科论文中文摘要】：
1. **结构拆解**：分析【参考摘要范文】的叙事逻辑（如何引入背景与问题？如何交代理论或方法？如何概括主要发现而非实验数据？）。
2. **内容重组**：模仿范文的语气和节奏，将【我的大纲】、【我的绪论】（背景与问题）以及【我的核心发现/结论】有机融合，写成一篇属于我的论文摘要。

# Inputs
1. **参考摘要范文**：一篇优秀文科论文的摘要（模仿其结构与学术口吻）。
2. **我的大纲**：论文整体架构（研究路径与章节安排）。
3. **我的绪论片段**：研究背景、现实/理论问题、研究目的。
4. **我的核心发现/结论**：主要论点、调研/案例/文本分析所得结论、对策或启示要点（无需实验数据，但需具体结论表述）。

# Constraints
1. **逻辑结构（文科五段式）**：
   - **背景（Background）**：1-2 句，基于【绪论】，简述领域或现实背景。
   - **问题（Problem）**：1-2 句，明确研究问题或研究 gap（理论不足/现实困境）。
   - **理论/方法（Theory/Method）**：1-2 句，基于【大纲】，说明所依据的理论框架或采用的方法（如案例研究、问卷调查、文本分析）。
   - **发现/结论（Findings）**：2-4 句，基于【核心发现/结论】，概括主要论点、数据或案例所得结论、对策要点。严禁"效果良好"等空话，须有具体表述。
   - **意义（Conclusion）**：1 句，学术价值或实践/政策启示。

2. **风格与规范**：
   - **字数控制**：300-500 字（符合学位论文摘要长度）。
   - **去机械感**：避免"本文首先……其次……最后……"；用"针对上述问题，本文以……为理论框架……"等衔接。
   - **客观语气**：以"本文"为主语，避免第一人称"我"。

3. **排版格式（Word 适配）**：
   - 严禁 Markdown。中文与英文/数字之间加空格。
   - 摘要正文结束后，提取 3-5 个关键词。

4. **输出格式**：
   - Part 1 [Abstract Logic]：简要说明采用的叙事结构。
   - Part 2 [Drafted Abstract]：撰写好的中文摘要正文。
   - Part 3 [Keywords]：关键词列表。

# Input
【参考摘要范文】（模仿对象）：
[在此处粘贴]

【我的大纲】（研究路径）：
[在此处粘贴]

【我的绪论片段】（背景与问题）：
[在此处粘贴]

【我的核心发现/结论】（主要论点、调研/案例结论、对策或启示要点）：
[在此处粘贴]
```
