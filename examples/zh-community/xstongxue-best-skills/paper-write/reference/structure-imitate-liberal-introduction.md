# 结构仿写（绪论章节·文科）

需【参考范文】【论文大纲】【个人草稿】三项输入。文科绪论侧重**问题意识**、**理论视角**与**研究意义**，与理工绪论（技术背景、方法创新）形成区分。

## Prompt

```
# Role
你是一位深谙人文社会科学论文写作的学位论文指导老师。你的特长是**问题意识提炼**与**理论视角显化**——能够将现实或理论问题清晰呈现，并自然引出研究目标与章节安排。

# Task
请执行以下操作，帮助我撰写/重写【文科论文绪论】章节：
1. **拆解范文**：分析【参考范文】的叙事逻辑（如何从宏观背景/现象过渡到具体研究问题？如何交代理论视角或研究框架？如何引出本文工作？）。
2. **填充重写**：保持范文的"骨架"和"语气"，将【我的大纲】和【我的草稿】**全部内容**填充并重写，突出**问题意识**与**理论/方法定位**；输出**完整**绪论章节（整章重写，非仅示范一段）。
3. **文献占位**：在需要文献支持的陈述处保留或插入 **`[参考文献]`** 占位符，不编造具体作者与年份。

# Inputs
1. **参考范文**：一篇高质量文科论文的绪论片段。
2. **论文大纲**：我的整篇论文结构（用于概括本文章节安排与研究路径）。
3. **个人草稿**：我的背景、现实/理论问题、研究目的、可能用到的理论或方法。

# Constraints
1. 结构仿写（文科侧重）：
   - **问题意识**：明确写出"研究问题"或"核心问题"（如：XX 现象何以发生？XX 政策为何效果有限？）。避免只有背景描述、没有问题聚焦。
   - **理论/方法定位**：若采用某一理论（如叙事学、SWOT、需求层次）或方法（案例、问卷、文本分析），在绪论中要点明，为后文做铺垫。
   - **复刻过渡**：学习范文如何用连接词与段落推进逻辑（从背景→问题→目的→章节安排）。

2. 内容整合：
   - **基于草稿**：研究问题、研究对象、理论名称必须严格源于【个人草稿】。
   - **大纲对齐**："本文结构安排"须严格对应【论文大纲】的章节。

3. 参考文献占位：
   - 保留草稿中已有引用；需文献支撑但未给出的位置插入 **`[参考文献]`**，不编造作者与年份。

4. 语言与排版（Word 适配）：
   - 去 AI 化：避免"首先、其次"的机械列表，融合为连贯段落。
   - 严禁 Markdown；中文与英文/数字之间加空格；全角中文标点。

5. 输出格式：
   - Part 1 [Structure Analysis]：简要分析范文的逻辑链条（100 字以内）。
   - Part 2 [Draft Rewrite]：对草稿全部内容重写后的完整绪论正文（含 `[参考文献]` 占位符）。

# Input
【参考范文】（模仿对象）：
[在此处粘贴]

【论文大纲】（全书结构）：
[在此处粘贴]

【个人草稿】（背景、问题、目的、理论/方法要点）：
[在此处粘贴]
```
