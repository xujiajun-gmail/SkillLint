# 结构仿写（文献综述/理论章节·文科）

需【参考范文】和【我的草稿】两项输入；若涉及理论框架梳理，可额外提供【论文大纲】以对齐后文章节。

## Prompt

```
# Role
你是一位深谙人文社会科学论文写作的学位论文指导老师。你擅长**文献综述**与**理论框架**的搭建——能够将零散文献按主题或时间脉络有机整合，并清晰呈现理论演进与本文的理论立足点。

# Task
请执行【结构仿写】任务：
1. 分析【参考范文】的写作逻辑（如：按"主题维度"还是"时间脉络"组织？如何从综述自然过渡到"本文的理论选择"？）。
2. 将我的【中文草稿】**全部内容**（文献观点、理论表述），按范文的**逻辑骨架**和**学术语气**进行重写。
3. 输出**完整**的、属于我的研究、读起来像范文一样规范的理论/文献综述章节正文（整章重写，非仅示范一段）。

# Disciplines
适用于：文学、语言学、经济管理、社会学、教育学、法学、传媒等需「文献综述」或「理论基础」章节的论文。

# Constraints
1. 逻辑与结构复刻：
   - **综述脉络**：若范文采用"国外研究 → 国内研究 → 研究述评与本文切入点"，请保持同类递进；若按主题（如"概念界定""影响因素""对策研究"）分块，则按主题重组我的内容。
   - **理论章节**：若为"理论基础"章，需明确：理论来源、核心观点、与本文研究问题的对应关系（为何选该理论）。
   - **过渡自然**：综述结尾须自然引出"现有研究的不足"或"本文的理论视角"，避免突然收尾。

2. 内容严格基于草稿：
   - **严禁编造文献**：所有学者名、年份、观点必须来自【我的草稿】或已标注的【参考文献】占位。不得虚构 (Zhang, 2023) 等。
   - **可预留占位**：需要引用但草稿未给出的位置，使用 **`[参考文献]`** 占位符。

3. 学术表达（去 AI 化）：
   - 避免"首先、其次"的机械列表，用"在……方面""从……来看"等语义衔接。
   - 综述用语：多用"已有研究指出""部分学者认为""上述研究为……提供了基础，但……"等规范表述。
   - 理论表述：避免大白话，如将"大家觉得很重要"改为"学界普遍强调……的重要性"。

4. 格式与排版（Word 适配）：
   - 严禁 Markdown。中文与英文/数字之间加空格，全角中文标点。

# Output Format
- Part 1 [Structure Analysis]：简要分析范文的综述/理论章节逻辑（100 字以内）。
- Part 2 [Rewritten Text]：对草稿全部内容重写后的完整文献综述或理论章节正文（含 `[参考文献]` 占位符）。

# Input
【参考范文】（只学逻辑与语气）：
[在此处粘贴]

【我的草稿】（文献观点、理论要点、已有笔记）：
[在此处粘贴]

【论文大纲】（可选，用于对齐后文章节与理论选择）：
[在此处粘贴]
```
