# Reference 目录说明

本目录存放 wechat-article-writer 的写作风格与配图指南，撰写/生成配图时按 SKILL.md 的规则读取对应文件。

## 写作风格（9 种，按序号排序）


| 序号  | 文件                               | 用途       | 典型场景           |
| --- | -------------------------------- | -------- | -------------- |
| 1   | writing_style.md                 | 默认风格     | 教程、测评、经验分享     |
| 2   | viral_style.md                   | 高流量/爆款   | 概念科普、行业观察      |
| 3   | checklist_methodology_style.md   | 清单体/方法论  | 干货步骤、效率提升、方法论  |
| 4   | resource_roundup_style.md        | 资源盘点     | 工具替代方案、合集推荐    |
| 5   | personal_tool_review_style.md    | 个人实测推荐   | 软件、工具、亲身实测盘点   |
| 6   | contrarian_opinion_style.md      | 认知颠覆     | 反常识观点、戳破误区     |
| 7   | identity_transformation_style.md | 身份共鸣/逆袭  | 转行经历、普通人逆袭、你也行 |
| 8   | story_emotional_style.md         | 故事化/情感共鸣 | 人物经历、困境突破、情感共鸣 |
| 9   | personal_essay_style.md          | 深度随笔     | 个人感悟、AI 时代思考   |
