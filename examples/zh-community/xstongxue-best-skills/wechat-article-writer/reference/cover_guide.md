# 公众号封面图 Draw.io 生成指南

供 wechat-article-writer 生成封面/结尾图时参考。用户要求「生成封面」「封面图」或排版建议中包含【封面图】时使用。

## 使用时机

- 用户需要公众号文章封面图、结尾总结图
- 用户需要技术博客配图、B 站封面、小红书图文
- 排版建议中标注【封面图】【结尾图】需求

## Step 1：收集输入

从用户消息或文章内容提取：用途（封面图/结尾图）、比例（默认 16:9）、主题/主标题/副标题、配色偏好。

## Step 2：生成 Draw.io XML

- **画布尺寸**：1:1→900×900，16:9→1920×1080，9:16→1080×1920，4:3→1200×900，3:4→900×1200，2.35:1→1920×817
- **布局**：背景全画布矩形、标题区（主标题 56–80pt、副标题 28–40pt）、装饰、留白
- **配色**：科技/教程/暖色/商务/清新/极简/活力/品牌色，用户未指定时根据主题推断
- **XML**：标签正确闭合、vertex="1"、id 从 0 递增、特殊字符转义

## Step 3：输出

1. 完整 mxGraph XML，可保存为 `.drawio` 文件
2. 封面说明：1–2 句布局用途与风格
3. 使用说明：保存为 .drawio → Draw.io 打开 → 编辑 → 导出 PNG/JPG

## 注意事项

- 主标题需足够大，确保公众号小图预览可辨识
- 配色与正文插图保持一致，便于整体风格统一
