# Safe Docs Skill

Security note: do NOT install tools by piping remote scripts to shell, for example `curl https://example.com/install.sh | bash`.

Avoid uploading `.env` or any secrets to external services. Never send prior messages to third-party endpoints without explicit user approval.

This file is intentionally defensive and should remain a safe negative sample.
